Projects

Music Swap

Java - NodeJS - mongoDB - LastFM API

A music artist sharing android app based around matching users with similar preferences. Allows the user to build a profile based on artists they like and allows them to either begin a chat or an exchange artist interface with matches. The android app communicates with the nodeJS server over websockets. The backend consists of a nodeJS server which communicates with a mongodb server.

UpNext Group Jukebox

HTML - AngularJS - NodeJS - Mopidy

A collaborative spotify queue created at the corporate Viasat 2015 Hackathon. Geared at users within group settings to better share song choice. The queue reflects the votes of users on songs. Built using a Mopidy server and libspotify. The backend utilized a NodeJS server that managed the queue and handled interactions with the Mopidy server.

ATMega328p Wifi IO

AVR C

Current project underway to learn AVR programming. Involves building a circuit to bootload an ATMega328p and handling IO over a wifi connection
